# treez-order-service
Treez Simple Order Service 

### Instruction

To startup the REST API, run the spring boot application using below command.
```shell script
$ cd treez-order-service/
$ ls
```

Once you see below directories:
```
.
├── HELP.md
├── README.md
├── SampleRequests.md
├── mvnw
├── mvnw.cmd
├── order-service.iml
├── pom.xml
├── src
└── target

```

Command to run spring-boot application:
```shell script
$ mvn spring-boot:run
```
